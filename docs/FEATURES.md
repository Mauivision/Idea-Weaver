# Idea Weaver — Feature Guide

## Analytics Dashboard

**Access:** Press `4` or click "Analytics" in view mode selector

- Total ideas, favorites, connections, notes stats
- Top 5 categories and tags
- Most connected ideas, recent activity timeline

## Advanced Search

**Access:** List view (top of page)

- Text search (titles, descriptions, tags, notes)
- Category, tag, connections, favorites filters
- Date range filter
- Active filters badge, "Clear All"

## Bulk Operations

**Access:** Ctrl+Click to select ideas

- Bulk edit category
- Bulk add tags
- Bulk delete with confirmation

## Data Export/Import

**Access:** Floating button (bottom-right)

- Export: JSON (recommended), CSV
- Import: JSON, validates and merges

## Keyboard Shortcuts

**Access:** Press `?` anywhere

- `Ctrl+K` — Focus search
- `Ctrl+Z` — Undo | `Ctrl+Y` — Redo
- `Ctrl+S` — Manual save
- `1`–`4` — Switch views
- `Esc` — Close dialogs

## Undo/Redo

- 50-action history, header indicators, keyboard shortcuts

## Idea Templates

**Access:** Templates view mode

- Default templates (Project Idea, Feature Request, Bug Report, etc.)
- Custom templates (create, save, delete)
- Persist in localStorage

## Smart Linking

- Content similarity analysis, one-click connect
- Appears in List view when multiple ideas exist

## Auto-Save

- 2-second debounce, background save, no manual save required

## Voice Input

- Web Speech API, floating mic (bottom-left)
- Interim + final transcription, error handling

## Notes Board

- Add notes via + FAB, drag to reposition, delete with confirmation
- Snap-to-grid, dark mode support
